# Bank Management System
## Project Description
This is a simple Bank Management System developed using C++.  
The project allows the user to manage bank clients by viewing, adding, deleting,searching, and updating client records.  
All data is stored using file handling.

## Features
- Show Clients. 
    Display all clients. 
- Add Client  
    Add new client.  
- Delete Client  
     Remove client.  
- Update Client  
     Edit client data. 
- Find Client  
      Search for client.

- C++
- STL (Vector) 
- functions
Files Handling
structures.
